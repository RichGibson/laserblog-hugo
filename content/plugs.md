I got a splitter for the dryer outlet, so I can use the same outlet for the 
laser and the dryer. I can probably use them both at the same time. That outlet
is provisioned for a lot of power!

I made my 220 to normal outlet pigtail, because the laser is running 220 over
normal outlets. 

And I ordered a splitter, so both the dryer and laser could be plugged in at
the same time.

But oddly, the outlets on the splitter were really stiff.

![A loose connection](/images/tighten_plug-loose_plug.jpg)

But a large c-clamp can be used to tighten that right down!

![Tightening the plug](/images/tighten_plug_c-clamp.jpg)
