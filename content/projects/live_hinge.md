I have a directoy named 'laser_box' with two files in it.

It is a parametric model for a live hinge. Specify a few parameters 
in the OpenSCAD file and you get a live hinge that you can export to 
a dxf, and then read into your laser software.

![a live hinge](/images/boxes/live_hinge.png)

- [hinge_work.dxf](static_files/boxes/laser_box.dxf)
- [hinge_work.scad](static_files/boxes/laser_box.scad)

A live hinge is a single piece of material, but cut such that it will
move like a hinge.

They work well for making rounded boxes, but mostly the material doesn't
work for that long if you use it as a hinge.

The files are dated Dec 7, 2016. So sometimes things stick around for a bit.
