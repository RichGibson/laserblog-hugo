Inkscape can convert many files.

Install Inkscape

Set a symlink to the exectable

sudo ln -s /Applications/Inkscape.app/Contents/MacOS/inkscape /usr/local/bin/inkscape

inkscape -o clothespin_jig_x5.svg clothespin_jig_x5.cdr
